User Guide

Master your application launcher. Configure, connect, and control your PC remotely with glassmorphism UI.

Getting Started

1

Launch Application

Double-click aviator-wails.exe to start the desktop application.

The glassmorphism interface will appear automatically!

2

Add Applications

Click + Add App button, then use the native file picker to select any .exe file.

App name auto-fills from filename!

Mobile Access

Instant QR Connection

The fastest way to connect from your phone. Zero configuration required.

  1. Look at the Status Panel in the desktop app (top section).
  2. Open your phone's Camera app.
  3. Scan the QR Code displayed in the app.
  4. Tap the notification link to open the web interface.
  5. Launch apps from the beautiful mobile UI!

Scan Me

Network Requirement

Your Phone and PC must be on the Same Wi-Fi Network for mobile access.

Advanced Features

Launch Arguments

Add flags like -windowed or --incognito in the "Arguments" field when adding/editing apps.

Native File Picker

Click Browse button to use Windows native file dialog for selecting executables.

Auto-Save

All changes are automatically saved to %LOCALAPPDATA%\Aviator\config.json

Pro Tips

  • Pin the app to taskbar for quick access
  • Use the web interface on tablets for a bigger screen
  • Add Steam games with -bigpicture flag
  • Launch browsers with --new-window for clean sessions
