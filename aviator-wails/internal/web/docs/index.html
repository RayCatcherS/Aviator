Introduction

Aviator v2.0 - Built with Wails

Aviator is a modern desktop application launcher with glassmorphism UI, built using Wails framework. Launch your Windows applications remotely from any device on your local network with a beautiful, native interface.

Launch Remotely

Trigger games, productivity tools, or scripts on your PC directly from your phone's browser or native desktop interface.

Zero Config

Automatic discovery via mDNS/Zeroconf. No IP typing required - just scan the QR code and launch!

Glassmorphism UI

Modern, premium interface with blur effects, smooth animations, and vibrant gradients that look amazing on any device.

Custom Arguments

Launch apps with specific flags (e.g. -bigpicture, --incognito) for tailored execution.

Native Performance

Built with Wails v2 and Go - native performance with small footprint (~12 MB executable).

Cross-Platform Ready

Wails framework enables future macOS and Linux builds from the same codebase.

What's New in v2.0

Wails Framework: Migrated from lxn/walk to Wails for better cross-platform support
Glassmorphism UI: Complete redesign with modern glass effects and animations
Vue 3 Frontend: Modern reactive components with smooth transitions
Improved Developer Experience: Live reload, browser debugging, TypeScript bindings

Quick Start

1

Download aviator-wails.exe from the releases page.

2

Double-click to run the application.

3

Click "Add App" in the desktop interface to configure your applications.

4

Scan the QR code with your phone to access the web interface!

System Requirements

  • Windows 10/11 (WebView2 included)
  • ~12 MB disk space
  • Local network connection (for mobile access)
  • Optional: For development - Go 1.23+, Node.js 18+
