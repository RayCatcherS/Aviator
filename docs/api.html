API Reference

The Aviator internal server runs on port 8000 by default.

Authentication

If enabled, all API endpoints (except /api/info and /api/auth) require a valid session.

  • Method: HttpOnly Cookie (aviator_key)
  • Session Duration: 24 hours

Endpoints

GET /api/info

Returns server status and authentication requirements. Does not require authentication.

Response

{
  "status": "running",
  "version": "v2.8.2",
  "hostname": "DESKTOP-XXXX",
  "auth_required": true,
  "is_authorized": false
}
POST /api/auth

Validates PIN and sets the session cookie.

Request Body

{ "pin": "123456" }

Response

200 OK
GET /api/apps

Lists all configured applications.

Response

[
  {
    "id": "uuid...",
    "name": "Firefox",
    "path": "C:\\Program Files\\...",
    "icon": "base64..."
  }
]
POST /api/launch/{id}

Launches the specified application on the host machine.

Response

{
  "status": "success",
  "pid": 1234
}
