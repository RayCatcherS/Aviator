REST API Endpoints

The server listens on port 8000 by default. All responses are JSON.

GET /api/apps

Retrieves the registry of all configured applications.

Response 200 OK 
[
  {
    "id": "550e8400-e29b...",
    "name": "Steam",
    "path": "C:\\Program Files\\Steam\\steam.exe",
    "args": "-bigpicture"
  }
]
POST /api/launch/{app_id}

Executes the application identified by ID on the host machine.

200 OK Launched successfully
404 Not Found Invalid ID or file missing

Codebase

main.py

Role: Entry Point & Thread Orchestrator

  • Spawns uvicorn in a daemon thread.
  • Runs tk.mainloop() in the main thread.
  • Handles graceful shutdown via os._exit(0).

launcher.py

Role: Execution Engine

Uses subprocess.Popen for non-blocking process creation.

Popen(cmd, cwd=dir)

config.json

Role: Persistence

Simple JSON array structure. Stored in %LOCALAPPDATA%/Aviator/config.json.
