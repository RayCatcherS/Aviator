User Guide

Master your local network launchpad. Configure, connect, and control your PC remotely with the Wails-powered desktop app.

Getting Started

1

Download & Launch

Download aviator-wails.exe from the releases page and double-click to launch.

The application initializes the local server environment.

2

Add Applications

Click Add App in the desktop window, then use the file picker to select any .exe file from your system.

3

Start Server

Click Start Server to activate the HTTP server. The QR code and network URLs appear automatically.

4

Monitor & Connect

Scan the QR code to open the dashboard. View running apps (Green LED) or stopped apps (Grey LED) in real-time.

Desktop Interface

Main Window Features

App Management

  • • Add App: Native file dialog to select executables
  • • Edit App: Modify name, path, or launch arguments
  • • Remove App: Delete apps from your list
  • • App List: View configured apps and their current status (Running/Stopped)

Server Control

  • • Start/Stop: Control the HTTP server with one click
  • • Status Indicator: Green (Running) or Red (Stopped)
  • • Local URL: http://localhost:8000
  • • Network URL: Your PC's IP address on the local network

Settings & Preferences

  • • Run at Startup: Automatically launch Aviator securely on Windows login
  • • Settings Persistence: Configurations stored in settings.json

Connecting Mobile

Instant QR Connection

The fastest way to connect. No typing required.

  1. Ensure the server is Running
  2. Look at the QR Code in the desktop app (right side)
  3. Open your phone Camera app
  4. Scan the QR Code
  5. Tap the notification link to open the Web App

Scan Me

Network Requirement

Your Phone and PC must be connected to the Same Wi-Fi Network. Corporate or guest networks may block mDNS discovery.

Advanced Features

Launch Arguments

Add custom flags when editing an app. Examples: -windowed, --incognito, -bigpicture.

System Tray

The app minimizes to the system tray when closed. Right-click the tray icon to stop the server or exit.

Safe Shutdown

Click Stop Server before closing to ensure proper cleanup. Exit via tray menu prompts for confirmation.

Troubleshooting

Can't connect from phone

  • • Verify both devices are on the same WiFi network
  • • Check Windows Firewall isn't blocking port 8000
  • • Try typing the Network URL manually instead of scanning QR
  • • Restart the server using Stop/Start buttons

App won't launch

  • • Verify the .exe path is still valid (not moved/deleted)
  • • Check launch arguments for typos
  • • Try launching the app directly from Windows first
  • • Some apps require admin rights—run Aviator as administrator

Desktop app won't start

  • • Ensure Windows 10/11 with WebView2 installed (usually pre-installed)
  • • Download WebView2 Runtime from Microsoft if needed
  • • Check antivirus isn't blocking the executable
  • • Try running as administrator
