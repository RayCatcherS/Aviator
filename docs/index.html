Introduction

Aviator is a desktop application that enables remote execution of Windows applications from devices on the local network, utilizing Wails for backend-frontend communication.

Launch Remotely

Control your PC applications from your phone's browser. Perfect for launching games, tools, or scripts from anywhere in your home.

Zero Config

Automatic discovery via mDNS/Zeroconf. No IP typing required—just scan the QR code and you're connected.

Responsive Interface

Adaptive user interface built with Vue 3, ensuring usability across desktop and mobile viewports.

Native Performance

Built with Go and Vue3. Fast, lightweight (~12 MB), with native file dialogs and system integration.

Quick Start

1

Download aviator-wails.exe from the releases page.

2

Launch the application—the interface opens automatically.

3

Click "Add App" to add your favorite Windows applications.

4

Start the server and scan the QR code with your phone!

Desktop App

Native Windows application built with Wails, Go, and Vue 3. Includes system tray integration and process management.

  • • Manage applications via file dialogs
  • • Control HTTP server lifecycle
  • • Display connection details (QR/URL)
  • • Background execution via system tray
  • • Auto-start with Windows

Mobile Web Client

Lightweight web client served by the internal HTTP server. Provides remote control capabilities via REST API.

  • • Status monitoring of remote processes
  • • Execution triggers via touch interface
  • • Responsive layout adaptation
  • • Cross-browser compatibility
