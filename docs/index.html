Introduction

Aviator is a modern desktop application that lets you launch Windows applications remotely from any device on your local network, powered by Wails and featuring a beautiful glassmorphism UI.

Launch Remotely

Control your PC applications from your phone's browser. Perfect for launching games, tools, or scripts from anywhere in your home.

Zero Config

Automatic discovery via mDNS/Zeroconf. No IP typing required—just scan the QR code and you're connected.

Premium UI

Glassmorphism design with smooth animations and blur effects. Native desktop experience powered by Wails.

Native Performance

Built with Go and Vue3. Fast, lightweight (~12 MB), with native file dialogs and system integration.

Quick Start

1

Download aviator-wails.exe from the releases page.

2

Launch the application—the interface opens automatically.

3

Click "Add App" to add your favorite Windows applications.

4

Start the server and scan the QR code with your phone!

Desktop App

Native Windows application built with Wails, Go, and Vue 3. Features modern glassmorphism design and system tray integration.

  • • Manage applications with native file dialogs
  • • Start/Stop HTTP server
  • • View QR code and network URLs
  • • Runs in system tray when minimized

Mobile Web

Beautiful web interface accessible from any device on your network. Optimized for mobile browsers with touch interactions.

  • • Glassmorphism cards with animations
  • • Touch-optimized launch buttons
  • • Responsive design for all screen sizes
  • • Works on iOS, Android, and desktop browsers
