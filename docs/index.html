Overview

Aviator is a local network application launcher and process monitor. It allows you to launch and monitor Windows applications remotely from any device on your local network via a secure internal HTTP server.

Remote Execution

Launch executables on the host machine via authenticated HTTP requests from mobile or other desktop devices.

Process Monitoring

Real-time status tracking of configured applications using native Windows process APIs.

Security

PIN-based authentication with HttpOnly cookies ensuring secure access control within the local network.

Tech Stack

Built with Go (Backend/Server), Wails (Bindings), and Vue 3 (Desktop UI).

Quick Start

1

Download the latest release.

2

Run aviator-wails.exe (Portable, no install required).

3

Add applications via the "Add App" button.

4

Start the server and access the dashboard via the displayed QR code.
