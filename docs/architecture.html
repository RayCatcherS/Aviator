System Architecture

WebApp Aviator operates on a strictly local loop. No data leaves the network.

Core Components

The Host (Server)

Backend
  • FastAPI
    High-performance async server handling REST requests.
  • Tkinter GUI
    Lightweight management interface running on the main thread.
  • Zeroconf (mDNS)
    Broadcasts `_aviator._tcp` for auto-discovery.

The Client (Web)

Frontend
  • Vanilla Stack
    Pure HTML/JS. No Node.js build steps required. Served as static files.
  • Tailwind CSS
    Utility-first styling loaded via CDN for rapid UI development.

Why this architecture?

By serving the frontend directly from the Python backend, we eliminate the need for users to install Node.js or run a separate frontend server. This creates a true "Single Executable" experience perfect for non-technical users.
