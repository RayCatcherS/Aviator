System Architecture

A modern hybrid architecture combining native Go performance with Vue 3 reactivity, bridged by Wails.

Data Flow

Aviator combines a native desktop application with a web server for remote access. All data stays within your local network.

Core Components

Desktop Application

Wails Frontend
  • Vue 3
    Reactive UI with composition API and modern component architecture.
  • Glassmorphism CSS
    Custom design system with gradient backgrounds, blur effects, and smooth transitions.
  • QR Code Generation
    Instant mobile connection via scannable QR codes.

Go Backend

Wails Backend
  • HTTP Server
    Embedded server serving static files and REST API endpoints.
  • Config Manager
    JSON-based persistence in %LOCALAPPDATA% with thread-safe operations.
  • mDNS Discovery
    Auto-discovery via Zeroconf (_aviator._tcp service).

Wails Bridge

Wails creates a seamless connection between the Vue frontend and Go backend without HTTP overhead.

Key Features

Type-Safe Bindings

Auto-generated TypeScript definitions for all Go methods. Full IDE autocomplete and type checking.

Native Dialogs

Use system file/folder pickers directly from JavaScript. No web input hacks needed.

Event System

Bi-directional event emitting. Backend can notify frontend of server status changes.

Single Binary

Vue app is embedded in the .exe. No separate files to distribute or lose.

Mobile Web Interface

The embedded HTTP server serves a glassmorphism web app accessible from any device on the network.

Pure CSS

No build step required. Glassmorphism effects using CSS backdrop-filter and modern gradients.

Responsive

Touch-optimized cards that work beautifully on iOS, Android, tablets, and desktop browsers.

Fast

Vanilla JavaScript with smooth animations. No React bloat. Instant page loads.

Why Wails?

Wails combines the best of desktop and web development:

  • • Single Executable - No Python runtime, no Node.js dependencies. Just .exe
  • • Native Performance - Go's speed with Vue's reactivity. < 12 MB memory footprint
  • • Modern DX - Hot reload during development, TypeScript support, familiar Vue workflow
  • • Cross-Platform - Same codebase builds for Windows, macOS, and Linux
