User Manual

Installation

  1. Download: Get the latest release from the Releases page.
  2. Run: Execute aviator-wails.exe. No installation is required as it is a portable application.
  3. Prerequisites: Windows 10 or 11 (WebView2 is usually pre-installed).

Configuration

Adding Applications

  1. Click Add App in the top right corner of the window.
  2. Enter a friendly Name for the application.
  3. Browse for the Executable Path (.exe file).
  4. Optionally add Arguments (e.g., specific flags like --fullscreen).

Security Setup

By default, the internal HTTP server is accessible to anyone on your local network. It is highly recommended to set a PIN.

  1. Click the Settings Gear icon (top right).
  2. Select Modify PIN.
  3. Set a 4-6 digit PIN to enable authentication.
  4. This PIN will be required to access the web dashboard from other devices.

Remote Access

  • Ensure Server is Running Check for the green status indicator in the desktop app.
  • Scan QR Code Use your mobile device's camera to scan the QR displayed in the dashboard.
  • Manual Connect Alternatively, navigate to http://<YOUR_IP>:8000 in a web browser.

Troubleshooting

Server Offline

Click "Start Server" in the dashboard. Ensure port 8000 is not in use by another application.

Connection Failed

Ensure both devices are on the same Wi-Fi. Check Windows Firewall settings to allow aviator-wails.exe.

App Not Launching

Verify the path is correct and the user has permissions to execute the file.
